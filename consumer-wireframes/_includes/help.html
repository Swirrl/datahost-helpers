Guide to terms and other helpful info

Update model

Dataset Series

Top level wrapper to give a consistent name and base URI (location) to a set of data.

Release

A coherent set of data which follows a consistent structure/schema and methodology. A breaking change leads to a new release within the dataset series.

Revision

A named, ordered set of compatible change-sets, representing the 3 operations (append, delete and correction).

Operations

Append

The new observations/rows to 'logically add' to the next revision

Delete

The rows which should 'logically be removed' from the next revision

Correct/Revise

The set of rows which have been revised, corrected or amended. This is a special case of append/delete; where a new URI is automatically coined for the new row with a reference to the old observation which this change replaces.
